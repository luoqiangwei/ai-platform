我们需要让它把记忆存储在sqlite3数据库中。路径放在/data/ai_brain.db中吧！

里面包含几个表
分别为
表1: shot_memory
存储和谁相处的每次问答，包含时间等信息，并且
表2: core_memory
核心记忆，归纳聚合shot_memory，形成一段精炼的核心记忆。
同时别人和自己交流的态度等信息记录下来。
表3: task_memory
这张表记录了在和别人说话时，记录下要做的事情，结束时间，任务进度，下次自主出触发事件等信息
表4: agents
记录一些工具相关的信息
表5: skills
记录一些skills，包含用途、不能用在哪些场景等
表6: bootstrap
下一次自己做默写事情时，它的前置背景、注意事项、风险等
表7: headbeat
下一次激活任务的时间等信息
表8: identity
记录自己扮演什么样的身份，这个随时会变，每次变动都代表着自我认知发生了变化。
表9：soul
记录自己的一些核心信念
表10: tools
记录一些工具如何使用，比如ssh、相机、浏览器等
表11: user
记录自己对访问者的态度、想法等。

这些需要随着大模型的返回和需要进行携带和记录更新。
